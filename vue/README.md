## Implementation

This app is built with [webpack](https://webpack.github.io/) and [Babel](https://babeljs.io/).

To run this app;

- `npm install` if it's first time
- `npm run server`
- Open `http://localhost:8080`

To build `js/app.js`;

- `npm install` if it's first time
- `npm run build`
